Privacidade:

Dados que coletamos:

  • Nome de usuário
  • Senha (devidamente criptografada)
  • Elo
  • Data de cadastro (Dia/Mês/Ano)
  • Jogos jogados com outros jogadores cadastrados

Além dos resultados e movimentos das partidas entre jogadores, mais absolutamente nada é salvo nos servidores do Jeek Online, com relação a como você interage com outros jogadores ou com o site.

Não acredita? Você pode conferir nosso código-fonte, que é 100% open-source, no github para confirmar ;)

Segurança:

Por mais que seus dados sejam todos criptografados, é possível que um hacker possa descriptografá-los por meio de um ataque de força bruta, caso sua senha seja muito fraca.

Por isso, recomendamos que você tenha uma boa prática de senhas seguras. Para mais infomações sobre o assunto, recomendamos que você veja o vídeo: Como escolher uma senha - Computerphile
